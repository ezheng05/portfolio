Streamlit web app

Data visualization and sustainability analytics

Web app built with Streamlit for environmental monitoring and sustainability analytics.

Context: I conducted research as one of three undergraduate researchers in the Multimedia Commmunications Lab at BU. We deployed an environmental monitoring system across 8 campus buildings as part of BU's climate action initiative.

Skills: Streamlit, Python, CAD, MongoDB, data visualization, web development, sustainability analytics, data analysis.

Impact: Designed public-facing interface for visualizing real-time environmental data and energy usage, funded by Campus Climate Lab initiative (one of seven funded projects), installed sensors in 20+ classrooms, won Anthony Janetos Climate Action Prize (Boston University).

Details:

  • Designed custom sensor enclosures installed in 8 pilot classrooms, scaled to 20+ classrooms analyzing billions of data points.
  • Architected data pipeline using Python, Numpy, and Pandas to process real-time environmental readings while optimizing our MongoDB schema for fast queries.
  • Built Streamlit analytics platform where facilities managers can instantly identify current conditions and trends to improve air quality and energy efficiency.
  • Optimized time-series database with Plotly/Matplotlib visualizations.
  • Met with campus stakeholders to ensure platform delivered what they needed (e.g., custom date ranges, threshold analysis, multi-sensor comparisons).
  • Added capability for complex queries and interactive visualizations that translate environmental data into actionable decisions.

ReCo Digital web app

AI image recognition & data analytics

Web dashboard for waste tracking and analytics.

Context: ReCo Digital is a design and strategy consulting firm. I was one of two interns in charge of a sustainability project for a client. Our objective was to automate the quantification of food waste through receipt image recognition.

Skills: Python, computer vision, OpenAI API, Gemini API, data analytics, data visualization, web development, Flask, Supabase.

Impact: Delivered solution to client in 4 weeks, achieved 98% accuracy in receipt parsing, improved client's operational efficiency, reduced page load times by 40%.

Details:

  • Tackled multiple end-to-end sustainability-focused client projects.
  • Led product development: extracted receipt data with image recognition to calculate input-to-waste ratios.
  • Experimented with various LLM APIs and computer vision models to improve accuracy and efficiency.
  • Engineered data pipeline that categorizes foods into 12 groups and calculates total weights by category.
  • Evaluated feature trade-offs, applied prioritization frameworks, and implemented insights.
  • Leveraged Gemini API for receipt reading, OpenAI for data processing, Supabase/PostgreSQL for database management, and Flask/JavaScript/CSS for the interactive frontend dashboard.
  • Managed entire project lifecycle from technology selection through implementation to testing and deployment.
  • Presented solution to client, continually iterated to optimize user experience based on feedback and needs.
